How to setup Tier 1 Graft Network supernode on Ubuntu

Create VPS

Digital Ocean

Graft Repository

Graft Node

Graft Supernode

Download blockchain

  • mkdir -p $HOME/.graft/
  • cd $HOME/.graft/
  • curl http://graftbuilds-ohio.s3.amazonaws.com/lmdb.tar.gz | tar xzf -
  • cd lmdb && rm em* && rm lo*

Cmd for run and restore wallet with a remote node

  • ./graft-wallet-cli --restore-deterministic-wallet --daemon-address graft.observer:18981

Cmd for run restored wallet with a remote node

  • ./graft-wallet-cli --daemon-address graft.observer:18981

Cmd for stake transfer

  • stake_transfer [addr] [amount] [lockblocks](Number of blocks to lock the stake transaction for, max 5000) [sn_public_id_key] [sn_signature]

Supernode endpoint

  • http://*.*.*.*:18690/dapi/v2.0/cryptonode/getwalletaddress

Default supernode private key location

  • ~/.graft/supernode/data/supernode.keys

Screen

  • For create new screen: `screen -S node`
  • For exit from created screen: `Ctrl + a + d`
  • For back to created screen: `screen -r node`
