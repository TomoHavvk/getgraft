Supernode Stats

Path: https://getgraft.org/api/v1/supernode/stats

Return the stats which include number of active SN's, number of SN's per tier and monthly ROI per tier.

Inputs: None.

Outputs:

  • nodes - int; Number of online supernodes.
  • tiers - array; List of tiers.
    • tier - int; Number of tier.
    • nodes - int; Number of online supernodes in tier.
    • roi - double; ROI (monthly).

Example:

  
{
    "nodes": 1260,
    "tiers": [
        {
            "tier": 1,
            "nodes": 579,
            "roi": 2.9
        },
        {
            "tier": 2,
            "nodes": 332,
            "roi": 2.81
        },
        {
            "tier": 3,
            "nodes": 197,
            "roi": 2.84
        },
        {
            "tier": 4,
            "nodes": 152,
            "roi": 2.21
        }
    ]
}

Supernode List

Path: https://getgraft.org/api/v1/supernode/list

Return the list of supernodes

Inputs: None.

Outputs:

  • nodes - array; List of supernodes.
    • Address - string; Wallet address with required stake.
    • PublicId - string; Supernode public id.
    • StakeAmount - int; Stake amount.
    • StakeFirstValidBlock - int; Stake first valid block.
    • StakeExpiringBlock - int; Stake expiring block.
    • IsStakeValid - bool; Is stake valid.
    • BlockchainBasedListTier - int; Blockchain based list tier
    • AuthSampleBlockchainBasedListTier - int; Auth sample blockchain based list tier
    • IsAvailableForAuthSample - bool; Is available for auth sample.
    • LastUpdateAge - int; Last update age in secconds.
    • ExpirationTime - string; The remaining amount of time before the expiration in readable format.
    • IsOnline - bool; Is supernode online.
  • height - int; Current block height.
  • totalStake - int; Total GRFT in stake.
