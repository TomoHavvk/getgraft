Supernode Stats

Path: https://getgraft.org/api/v1/supernode/stats

Return the stats which include number of active SN's, number of SN's per tier and monthly ROI per tier.

Inputs: None.

Outputs:

  • nodes - int; Number of online supernodes.
  • tiers - array; List of tiers.
    • tier - int; Number of tier.
    • nodes - int; Number of online supernodes in tier.
    • roi - double; ROI (monthly).

Example:

  
{
    "nodes": 1219,
    "tiers": [
        {
            "tier": 1,
            "nodes": 592,
            "roi": 2.84
        },
        {
            "tier": 2,
            "nodes": 332,
            "roi": 2.81
        },
        {
            "tier": 3,
            "nodes": 217,
            "roi": 2.58
        },
        {
            "tier": 4,
            "nodes": 163,
            "roi": 2.06
        }
    ]
}
